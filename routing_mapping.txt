# Routing map for Trinity

The Cloudflare tunnel forwards `https://trinity.quantmatrixai.com` into the local host at `10.2.1.65` where Traefik dispatches requests by path:

- `/` &rarr; React frontend container on `10.2.1.65:8080`
- `/admin/` &rarr; Django backend on `10.2.1.65:8000`
- `/app/` &rarr; FastAPI backend on `10.2.1.65:8001`

## Django REST API
All Django API endpoints start with `/api/` once the request reaches the backend. Examples include:
- `/api/accounts/login/`
- `/api/accounts/logout/`
- `/api/accounts/users/me/`
- `/api/registry/...`
- `/api/subscriptions/...`
- `/api/workflows/...`
- `/api/atoms/...`
- `/api/atom-configs/...`
- `/api/config-store/...`
- `/api/permissions/...`
- `/api/orchestration/...`
- `/api/tenants/...`
- `/api/roles/...`
- `/api/audit/...`

## FastAPI endpoints
FastAPI lives under `/app/`. Key paths are:
- `/app/feature-overview`
- `/app/cards/archive`
- `/app/data-upload-validate`
- `/app/t/` for text box operations

Django REST endpoints use the `/admin/api/` prefix, for example `/admin/api/accounts/login/`.
